DESCRIPTION
========================================================

HybridBusinessPOS is a **production-ready, offline-capable web-based Business Management System** built using **C# ASP.NET Core**. 

The system is designed for **small and medium businesses**, including:

- Phone accessories shops
- Electronics stores
- Lighting shops
- Retail shops
- Mini supermarkets

It combines **POS (Point of Sale)** functionality with **cloud-ready SaaS features** and offline-first capabilities. The system can sync automatically when internet is available, making it suitable for regions with unstable connectivity.

INSTALLATION & RUNNING LOCALLY
========================================================

1. Clone the repository:
   git clone https://github.com/YourUsername/HybridBusinessPOS.git

2. Navigate to project folder:
   cd HybridBusinessPOS

3. Build the project:
   dotnet build

4. Run the project:
   dotnet run

5. Open in your browser:
   http://localhost:5000 or https://localhost:5001

6. For offline mode:
   - Use the web app normally
   - Sales and products will be stored locally in SQLite
   - Auto-sync occurs when internet connection is detected
